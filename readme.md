# Qariin

Segala puji bagi Allah SWT yang telah menurunkan Al-Qur’an sebagai petunjuk bagi manusia. Shalawat dan salam semoga tercurah kepada Nabi Muhammad SAW, keluarga, sahabat, dan seluruh umatnya hingga akhir zaman.

Dengan memohon ridha Allah, kami mempersembahkan sebuah aplikasi web sederhana yang menampilkan ayat-ayat suci Al-Qur'an lengkap dengan fitur **pemutaran audio per ayat**, agar memudahkan kaum muslimin dalam membaca dan mendengarkan kalamullah di mana saja.

## Fitur Utama

- Menampilkan seluruh surat dan ayat dari Al-Qur'an.
- Memutar audio per ayat menggunakan murottal dari qari Abdul Basit (Everyayah).
- Navigasi antar surat melalui dropdown.
- Pemutaran otomatis seluruh surat.
- Opsi **Ulangi Ayat** untuk membantu hafalan.
- Klik pada ayat untuk memutarnya secara individual.
- Klik ulang ayat yang sedang diputar untuk menghentikannya.
- Auto scroll ke ayat yang sedang dibaca.
- Tampilan bersih dan responsif (mobile-friendly).

## API yang Digunakan

- **[alquran.cloud](https://alquran.cloud/api)**  
  Digunakan untuk teks Al-Qur’an dan daftar surat.
- **[everyayah.com](https://everyayah.com)**  
  Digunakan untuk pemutaran audio per ayat.
